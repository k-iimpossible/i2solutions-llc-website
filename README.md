# NexusLabs — Agency Site

Next.js (App Router) + Tailwind CSS marketing site for a software development agency.

## Run locally

```bash
cd agency-site
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run start` — serve production build
- `npm run lint` — ESLint
